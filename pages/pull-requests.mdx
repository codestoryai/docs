# GitHub PR summary bot

While this is not (yet) a core part of the IDE, our summary bot leaves a summary with impact analysis on your Pull Requests.

![Sample summary](/images/prbot.png)

In order to try it out, navigate to the GitHub App page, [codestorybetabot](https://github.com/apps/codestorybetabot) and click _Configure_ to add it to your repository.
